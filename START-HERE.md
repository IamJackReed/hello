## üéØ Common Questions

**Q: Do I need to install anything?**
A: No! It's plain HTML, CSS, and JavaScript. Just open the files in a browser.

**Q: How do I edit the content?**
A: Open the HTML files in any text editor and change the text.

**Q: Can I change the colors?**
A: Yes! Edit the CSS variables at the top of `styles.css`.

**Q: How does the disclaimer work?**
A: Read `DISCLAIMER-GUIDE.md` for complete details.

**Q: How do I add a new blog post?**
A: Copy a template from the `posts/` folder, edit it, and add a card to the category page. See `GETTING-STARTED.md` for details.

**Q: Is this legally compliant?**
A: The disclaimer is designed for UK compliance, but you should review it with a solicitor for your specific situation.

**Q: How do I deploy this?**
A: Upload to any static hosting (GitHub Pages, Netlify, Vercel). See `README.md` for details.

**Q: Can I customize the design?**
A: Absolutely! Edit `styles.css` to change colors, fonts, spacing, etc.
