=====================================
DISCLAIMER FLOW
=====================================

User Journey for Investment Content:

1. User clicks "Market Diary" or investment post
   │
   ▼
2. JavaScript checks: Has user accepted disclaimer?
   │
   ├─ YES ──► Show content with banner
   │
   └─ NO ──► Show full-screen gate
              │
              ├─ "I Understand" ──► Save to localStorage
              │                     Show content with banner
              │
              └─ "Take Me Back" ──► Return to home page
